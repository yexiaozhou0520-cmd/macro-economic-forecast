# 宏观经济指标预测模型
**项目C｜量化金融/金工方向作品集**
中央财经大学｜大数据管理与应用

## 1. 项目简介
本项目基于机器学习构建**宏观经济指标预测系统**（重点预测CPI、GDP增长率），用于展示金融时间序列分析、量化建模与工程化代码能力。
项目具备完整可复现流程、规范代码结构、专业文档，符合海外硕士申请作品集标准。

模型使用月度宏观经济数据，通过随机森林回归完成预测，包含数据预处理、时序特征工程、模型训练验证、自动化图表输出全流程。

## 2. 项目结构
```text
macro-economic-forecast/
├── data/                 # 原始数据与处理后数据
│   ├── raw/              # 模拟原始宏观数据
│   └── processed/        # 清洗后用于建模的数据
├── models/               # 训练好的预测模型
├── results/              # 输出结果
│   └── figures/          # 趋势图、预测图、特征重要性图
├── src/                  # 核心代码
│   ├── main.py           # 主程序（一键运行）
│   ├── data_process.py   # 数据清洗与预处理
│   ├── model.py          # 模型训练与评估
│   └── visualize.py      # 可视化绘图
├── docs/                 # 项目文档
└── README.md             # 项目说明
```

## 3. 核心功能
- 自动生成5年月度模拟宏观经济数据
- 数据清洗：缺失值插值、异常值处理、时间格式标准化
- 时序特征工程：提取年份、月份等时间特征
- 基于随机森林回归的经济指标预测
- 模型评估：MAE、R² 等量化指标
- 自动生成专业可视化图表
- 输出标准预测报告

## 4. 使用技术
- Python 3.x
- Pandas / NumPy / Matplotlib
- Scikit-learn
- Joblib

## 5. 运行方式
1.  下载或克隆项目
2.  安装依赖库：
    ```bash
    pip install pandas numpy matplotlib scikit-learn joblib
    ```
3.  运行主程序：
    ```bash
    python src/main.py
    ```    
   图表、模型、报告将自动生成
